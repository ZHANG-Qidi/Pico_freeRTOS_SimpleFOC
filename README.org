* 项目简介
- 运行在 Pico C SDK 环境中的 Simple FOC 示例
- 通过修改 Arduino AVR Core, 提供一个基于 Pico C SDK 的 Arduino API 环境, 以运行 Simple FOC
** 项目进度
*** 已实现
- SPI 传感器读取
- I2C 传感器读取
- BLDC 扭矩(电压)控制
*** 未实现
- 电流采样
- 编码器
* Modules 源码说明
** Arduino-FOC-2.2
- Simple FOC 源码, 基本没有改动
** ArduinoCore-avr-1.8.7
- Arduino AVR Core 源码, 部分替换为 Pico C SDK, 提供一个基于 C SDK 的 Arduino API 环境
